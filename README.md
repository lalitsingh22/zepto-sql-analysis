# 🛒 Zepto SQL Data Analysis

## 📌 Overview
This project is a SQL-based data analysis case study on :contentReference[oaicite:0]{index=0} inventory and product data. The project focuses on analyzing pricing, discounts, stock availability, product categories, and estimated revenue insights using SQL queries and data analytics techniques.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Analyze product pricing and discount patterns
- Identify high-value and low-stock products
- Estimate revenue opportunities
- Generate category-wise business insights
